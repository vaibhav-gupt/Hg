Das RaumZeit Universum wird in einem [eigenen Mercurial 
Repository][RaumZeit_hg] entwickelt. 

[RaumZeit_hg]: http://bitbucket.org/ArneBab/raumzeit "Die Quelldateien des RaumZeit Universums."

Ihr könnt es einfach von dort klonen. Unter GNU/Linux und MacOSX via [Mercurial][] in 
der shell mit 

> hg clone http://freehg.org/u/inside/raumzeit

Unter Windows via [TortoiseHG][] von der gleichen Adresse. 

[Mercurial]: http://selenic.com/mercurial "Mercurial: Freie verteilte Versionsverwaltung"

[TortoiseHG]: http://tortoisehg.sf.net "TortoiseHG: Windows Explorer Integration für Mercurial."
