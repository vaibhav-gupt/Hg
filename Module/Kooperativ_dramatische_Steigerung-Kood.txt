Dieses Modul ermöglicht die Steigerung von Charakteren in dramatischen Szenen und fördert kooperatives dramatisieren von Szenen. 

Statt dass diejenigen Charaktere Punkte erhalten, die Großes vollbringen, erhalten diejenigen Cunkte, die die Taten erst großartig machen und dafür eigene Rückschläge in Kauf nehmen. 

Ihre Punkte nutzen können allerdings nur diejenigen, die große Szenen erleben. Sie können dabei ihre gesammelten Punkte als Bonus verwenden. Sobald sie Punkte als Bonus nutzen, sagen sie automatisch an, dass sie in dieser Szene steigern wollen, so dass sie keine Punkte sammeln können. Sie können das allerdings auch ansagen, ohne Punkte auszugeben. 

Eine Szene ist dabei ein abgeschlossener Handlungsrahmen, eben wie eine Szene in einem Film. Beispiele sind ein kurzer Kampf (Kämpfe sind in 1w6 schnell vorbei), das Gespräch mit einer NSC, eine Gruppeninterne Diskussion, usw. In einem Jacky Chan Film wäre jeder einzelne Kampfschauplatz eine eigene Szene, in Fairy Tail jede Szene mit abgeschlossener Gruppe von Charakteren. 

Wenn einer oder mehrere angesagt haben, dass sie in einer Szene steigern wollen, können die anderen Punkte sammeln, indem sie ihren Charakter auf eine Art scheitern lassen, die die Szene spannender macht und/oder sie stärker auf die steigernden Charaktere fokussiert. Es genügt nicht, den eigenen Charakter einfach aus der Szene zu nehmen. Die Handlung muss vielmehr die Szene für mindestens einen der Spieler der steigernden Charaktere intensiver machen. Dabei entscheiden einfach die steigernden Spieler, welche Handlungen die Szene gefördert haben. 

Nach der Szene kann der Spieler, der sie bestritten hat, Punkte zur Steigerung nutzen. 

Beispiel
--------

Fox, klar kalkulierender Pilot und Scharfschütze, sagt an, dass er steigern will, während sich Sskreszta und Kalem auf eine schwerbewaffnete Spezialeinheit stürzen.
